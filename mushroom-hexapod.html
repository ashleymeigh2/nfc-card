🍄 Mushroom Hexapod

Using fungal electrophysiology to control a walking robot.

Project Overview

I recently read the paper Sensorimotor Control of Robots Mediated by Electrophysiological Measurements of Fungal Mycelia, which demonstrated that the electrical activity generated by fungal mycelium can be used to control a robotic system. It inspired me to build my own version using an FNK0029 hexapod.

The aim of this project is to investigate whether the electrophysiological responses of oyster mushroom (Pleurotus ostreatus) mycelium can be used as a biological control interface for a hexapod robot.

How It Will Work

Rather than using a conventional controller, electrodes embedded within a living mycelium culture continuously record extracellular electrical activity. These tiny electrical signals are amplified, filtered and digitised before being processed by a microcontroller.

The mycelium is exposed to controlled light stimulation while its electrophysiological responses are recorded. Repeatable signal changes are translated into movement commands for the FNK0029 hexapod, allowing fungal electrical activity to influence robot behaviour.

Specific signal patterns may initiate forward movement, stop the robot or trigger turning behaviours, creating a simple biohybrid robotic system.

Experimental Workflow

  1. Hexapod Development – Configure the FNK0029 for autonomous locomotion and implement an interface capable of receiving external movement commands.
  2. Fungal Electrophysiology – Culture oyster mushroom mycelium and develop a reliable electrode interface for continuous extracellular recordings.
  3. Stimulus Characterisation – Measure the electrophysiological response of the mycelium to controlled light exposure, including response magnitude, latency, repeatability and the effect of varying light intensities.
  4. Biohybrid Integration – Integrate the fungal recording system with the robot controller, translating repeatable electrophysiological responses into real-time locomotion commands.

Project Goal

The objective is to demonstrate that light-induced changes in fungal electrical activity can reliably control a walking robot, exploring the intersection of biology, robotics and unconventional computing.
